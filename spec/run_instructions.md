# 🚀 Інструкція з запуску мультиагентної системи

Наступні кроки для запуску системи:

1. **Налаштування Telegram-бота**: 
   Додайте ваш токен `TELEGRAM_BOT_TOKEN` у локальний файл конфігурації `.env.local`.
   
2. **Запуск бази даних Neo4j**: 
   Переконайтеся, що ваш локальний сервер Neo4j запущено. Параметри підключення за замовчуванням (`bolt://localhost:7687` з базовими авторизаційними даними) вже прописані у конфігураціях.
   
3. **Запуск Streamlit Dashboard**: 
   Виконайте у терміналі скрипт запуску дашборду:
   ```bash
   ./run_dashboard.sh
   ```
   У дашборді з'являться нові вкладки:
   - **🤖 Агентне керування**: тут ви можете запустити бот-демон Telegram прямо з веб-інтерфейсу, спостерігати за планом виконання кроків та станом Orchestrator в реальному часі.
   - **📝 Імпорт тексту**: вікно для швидкого додавання та виправлення текстів.
